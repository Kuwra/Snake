Gra Snake napisana w C++ z biblioteką SFML.

Wymaga:
- CMake >= 3.11
- Kompilator C++ 
- SFML 2.6

Build:
- mkdir build && cd build
- cmake ..
- cmake --build . --config Release

Uruchomienie:
- ./Snake.exe # Windows
- ./Snake # Linux/macOS

Sterowanie:
- W / ↑ ; – ruch w górę
- S / ↓ ; – ruch w dół
- A / ← ; – ruch w lewo
- D / → ; – ruch w prawo

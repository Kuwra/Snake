Snake game written in C++ using the SFML library.

Requirements:
- CMake >= 3.11
- C++ compiler 
- SFML 2.6

Build:
- mkdir build && cd build
- cmake ..
- cmake --build . --config Release

Run:
- ./Snake.exe # Windows
- ./Snake # Linux/macOS

Controls:
- W / ↑ ; – move up
- S / ↓ ; – move down
- A / ← ; – move left
- D / → ; – move right
